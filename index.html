Projects

Autonomous Mapping of Dynamic Environments

Safely explore and map indoor areas with dynamic (human) actors using a KUKA YouBot and two infrared line-scanners running on ROS. Additionally, detect and name rooms found during exploration. 2 separate GUIs to control the robot: web-based and PyQt-based.

Live Demo: https://youtu.be/ECbo_HCrQ8U
Simulation Demo: https://youtu.be/neXzYKmijTU

Robotic Telepresence

A collaboration project between Aalto University and Futurice. Prototyping different approaches to telepresence using the open source robot InMoov, including Microsoft Kinect-based motion capture/replication (ROS) and physical prototypes for capturing arm/hand/head movements (IMUs / resistors + Arduino).

Unfortunately live footage of usage is lost in time.

3D Modeling Software

OpenGL / C++ practice project. A simple 3D modeling program with ability to create and view different shapes in different ways. 2D UI done with Dear ImGui, 3D rendering code is self-written.

Carla + Autoware integration

A ROS bridge between CARLA-simulator and Autoware (ROS) including using Autoware to map areas from LIDAR data and identifying vehicles in simulated RGB + LIDAR data using Autoware's SSD detector.
